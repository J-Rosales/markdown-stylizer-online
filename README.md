# Markdown Stylizer Online

Bootstrap documentation bundle for the **`markdown-stylizer-online`** repository.

This repository is intended to be hosted on **GitHub Pages** and developed as an **offline-capable** static single-page app for styling Markdown and exporting to **PDF (one-click)** and **PNG (ZIP, one per page)**.

## What’s in this bundle
- `docs/01-spec.md` — product + technical spec (MVP)
- `docs/02-architecture.md` — modules, state, rendering/export pipelines
- `docs/03-directory-layout.md` — recommended starting directory (copy/paste)
- `docs/04-milestones.md` — end-to-end milestone plan
- `docs/05-github-pages.md` — deployment notes (Pages + SPA routing)
- `docs/06-offline.md` — offline strategy, service worker, caching
- `docs/07-fonts.md` — Google Fonts support + offline/export limitations
- `docs/08-security.md` — strict Markdown, sanitization, threat model
- `docs/09-test-plan.md` — acceptance tests + manual QA checklist
- `docs/10-codex-brief.md` — short brief to paste into Codex as initial directive

## How to use
1. Create a new GitHub repo named **`markdown-stylizer-online`**.
2. Add the contents of this zip to the repo root.
3. Use `docs/10-codex-brief.md` as the initial prompt for your agent.
4. Implement milestones in `docs/04-milestones.md` sequentially.
